# 任务管理系统使用指南

## 快速开始

### 当前状态
记忆文件系统已重构完成，采用任务化管理模式。

---

## 目录结构概览

```
memory/
├── tasks/          → 任务目录（每个任务一个文件夹）
├── active/         → 当前活跃任务信息
├── raw-chat/       → 原始聊天记录归档
└── README.md       → 详细说明
```

---


## 使用命令

### 切换任务
- `/st <task_id>` - 切换到指定任务
- `/st` - 切换到聊天模式

### 更新任务
- `/ut` - 将聊天内容更新到任务文件

### 关闭任务
- `/ct` - 完成当前任务

### 查看任务
- `/ctask` - 查看当前任务信息

### 暂停任务
- `/deactivate_task` - 切换到无活跃任务状态
  - 在此状态下，记忆只同步到 `memory/YYYY-MM-DD.md`
  - 不会同步到任何 task 文件
  - 适用于：暂时暂停任务、进行其他工作、不需要任务跟踪时

## 示例场景

### 场景1: 创建建军小红书任务

```
/st jianjun-xiaohongshu
# 自动创建任务目录
# 开始记录相关对话
```

### 场景2: 暂时切换到聊天模式

```
/st
# 切换到 raw-chat-2026-03-04
# 记录日常对话
```

### 场景3: 更新任务内容

```
/ut
# 自动分析聊天内容
# 分类更新到 goal.md, context.md 等
```

### 场景4: 完成任务

```
/ct
# 标记任务为 completed
# 清空活跃状态
```
