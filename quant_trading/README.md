## 一、量化交易基础概念

### 1.1 什么是量化交易

**定义**: 用数学模型替代主观判断，通过计算机程序自动执行交易决策。

**核心要素**:
- **数据**: 行情、财务、另类数据（舆情、资金流向等）
- **模型**: 数学公式描述的选股/择时逻辑
- **回测**: 用历史数据验证策略有效性
- **执行**: 程序自动下单（或信号提醒+人工执行）
- **风控**: 仓位管理、止损止盈、风险敞口控制

**与主观交易对比**:

| 维度 | 主观交易 | 量化交易 |
|------|----------|----------|
| 决策依据 | 经验、直觉、消息 | 数据、模型、统计 |
| 执行方式 | 人工下单 | 程序自动/辅助 |
| 纪律性 | 易受情绪影响 | 严格执行规则 |
| 覆盖范围 | 只能盯几只票 | 可同时监控数百只 |
| 回测能力 | 难以验证 | 可用历史数据验证 |

## 二、A股量化交易特点

### 2.1 A股市场特殊性

**交易规则**:
- **T+1**: 当天买入次日才能卖出
- **涨跌停**: ±10%（主板）、±20%（创业板/科创板）、±5%（ST）
- **交易时间**: 9:30-11:30, 13:00-15:00
- **做空限制**: 融资融券门槛高，融券难

**市场特征**:
- **散户占比高**: 约70%交易量来自散户
- **情绪波动大**: 追涨杀跌明显
- **政策敏感**: 监管政策影响大
- **板块轮动快**: 热点切换频繁

**量化机会**:
- 散户非理性行为 → 均值回归策略
- 板块轮动 → 动量/轮动策略
- 财务数据滞后 → 基本面量化
- 舆情数据 → 事件驱动策略

### 2.2 A股量化主要限制

- **T+1限制**: 无法做日内高频，策略以波段为主
- **做空困难**: 多数策略只能做多
- **数据成本**: 高质量数据需要付费（如逐笔成交）
- **交易成本**: 印花税0.1%（卖出）、佣金万2-万3
- **滑点**: 涨跌停无法成交

## 三、量化策略分类

### 3.1 按持仓周期分

| 类型 | 持仓时间 | 特点 | A股适用性 |
|------|----------|------|-----------|
| 日内高频 | 分钟级 | 拼速度，T+0市场 | ❌ 不适用 |
| 日内波段 | 数小时 | 隔夜风险低 | ❌ 不适用(T+1) |
| 短线波段 | 1-5天 | 捕捉短期波动 | ✅ 主流 |
| 中线波段 | 1-4周 | 趋势跟踪 | ✅ 常用 |
| 长线投资 | 数月以上 | 基本面为主 | ✅ 基本面量化 |

### 3.2 按策略逻辑分

#### A. 技术指标策略
**原理**: 价格和成交量包含一切信息

**常用指标**:
- 均线系统（MA5/10/20/60交叉）
- MACD（趋势+动能）
- RSI/KDJ（超买超卖）
- 布林带（波动率通道）
- 成交量（放量/缩量）

**优点**: 简单易懂，信号明确
**缺点**: 滞后性，震荡市假信号多

#### B. 因子选股策略（多因子模型）
**原理**: 用多个因子打分，选高分股票

**常见因子类型**:

| 类别 | 具体因子 | 逻辑 |
|------|----------|------|
| 价值 | PE、PB、PS、PEG | 低估值的股票长期跑赢 |
| 成长 | ROE、营收增长、净利增长 | 高增长公司有溢价 |
| 质量 | 毛利率、ROA、现金流 | 好公司更稳健 |
| 动量 | 过去N月涨跌幅 | 强者恒强 |
| 波动 | 波动率、Beta | 低波动更稳 |
| 规模 | 市值 | 小盘股溢价（A股明显） |
| 技术 | 均线、RSI、换手率 | 技术面信号 |
| 资金 | 北向资金、主力净流入 | 聪明钱跟踪 |
| 舆情 | 新闻情绪、研报评级 | 市场预期 |

**构建流程**:
1. 因子挖掘（找有效因子）
2. 因子检验（IC、IR、分组回测）
3. 因子清洗（去极值、标准化、中性化）
4. 因子合成（等权/加权/机器学习）
5. 组合构建（打分→排序→选股）
6. 回测验证
7. 实盘运行

#### C. 统计套利策略
**原理**: 利用价格偏离统计规律的回归

**常见类型**:
- **配对交易**: 两只相关性高的股票，价差过大时做空高估、做多低估
- **板块轮动**: 追踪热点板块，配置强势板块
- **指数增强**: 相对指数做超额收益

**A股适用**: 配对交易较难（做空限制），板块轮动可行

#### D. 事件驱动策略
**原理**: 特定事件前后存在规律性机会

**常见事件**:
- 财报发布（业绩超预期）
- 分红送转（高送转）
- 股票回购
- 大宗交易
- 定增解禁
- 指数调整（入指/出指）
- 政策发布

**优点**: 逻辑清晰，可验证
**缺点**: 事件发生频率低，需要多事件组合

#### E. 机器学习策略
**原理**: 用ML模型学习价格预测

**常用模型**:
- 线性回归/逻辑回归
- 随机森林/XGBoost/LightGBM
- LSTM/Transformer（深度学习）
- 强化学习

**流程**:
1. 特征工程（因子→特征）
2. 标签构建（未来N日收益率）
3. 模型训练
4. 样本外验证（防过拟合）
5. 滚动训练+实盘

**优点**: 可挖掘非线性关系
**缺点**: 过拟合风险大，需要严格验证

## 四、策略开发流程

### 4.1 标准流程

```
[想法] → [数据准备] → [策略编写] → [回测] → [优化] → [模拟盘] → [实盘]
   ↓          ↓            ↓           ↓        ↓         ↓
 随时记录   数据清洗     逻辑实现    绩效评估  参数调优   小资金验证
```

### 4.2 回测关键指标

**收益指标**:
- **总收益率**: (期末-期初)/期初
- **年化收益率**: (1+总收益率)^(252/天数) - 1
- **超额收益**: 策略收益 - 基准收益

**风险指标**:
- **最大回撤**: 峰值到谷底的最大跌幅
- **波动率**: 收益率标准差（年化）
- **下行风险**: 只计算负收益的波动

**风险调整收益**:
- **夏普比率**: (年化收益-无风险利率) / 年化波动率
  - >1: 好，>2: 很好，>3: 优秀
- **索提诺比率**: (年化收益-无风险利率) / 下行风险
- **卡玛比率**: 年化收益 / 最大回撤

**胜率相关**:
- **胜率**: 盈利交易次数 / 总交易次数
- **盈亏比**: 平均盈利 / 平均亏损
- **期望收益**: 胜率 × 平均盈利 - (1-胜率) × 平均亏损

### 4.3 常见回测陷阱

| 陷阱 | 说明 | 解决方法 |
|------|------|----------|
| **未来函数** | 用了当时不知道的数据 | 严格时间对齐 |
| **过拟合** | 参数过度优化历史 | 样本外测试、交叉验证 |
| **幸存者偏差** | 只用现存股票数据 | 包含退市股票 |
| **前视偏差** | 用收盘价做日内决策 | 用当时可得的Open/High/Low |
| **交易成本低估** | 忽略滑点、冲击成本 | 加入合理的成本假设 |
| **涨跌停无法成交** | 假设能买到涨停股 | 添加涨跌停过滤 |

## 六、风险控制体系

### 6.1 仓位管理

**固定金额**: 每次买入固定金额
**固定比例**: 每次买入总资金的固定%
**凯利公式**: f* = (bp - q) / b（理论上最优，实际太激进）
**风险平价**: 按风险贡献分配仓位

### 6.2 止损止盈

**固定止损**: 亏损X%离场（如-5%）
**移动止损**: 价格上涨后，止损线上移
**ATR止损**: 用波动率动态设置止损距离
**时间止损**: 持仓N天不盈利则离场

### 6.3 分散化

- **标的分散**: 持有多只股票
- **策略分散**: 多策略组合
- **时间分散**: 分批建仓
