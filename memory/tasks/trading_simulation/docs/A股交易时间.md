# A股交易时间

- 收盘：11:30
- 交易时长：2小时

### 下午
- 开盘：13:00
- 收盘：15:00
- 交易时长：2小时

---

## 重要规则

### 1. 买入时间
- 只能在 9:30-11:30 或 13:00-14:57 之间买入
- 14:57之后禁止买入（离闭市太近）
- 11:30-13:00 为休市时间，不能交易

### 2. 卖出时间
- 只能在 9:30-11:30 或 13:00-15:00 之间卖出
- 15:00之前必须卖出或持仓过夜
- 模拟交易中，卖出时间应合理（14:50-14:59为宜）

### 3. T+1 规则
- 当日买入的股票，次日才能卖出
- 当日卖出后，次日才能买入同一股票
- 不能当日买卖同一股票（T+0）

---

## 模拟交易规则

### 时间设置
- 买入：必须设置在交易时段内
- 卖出：必须设置在交易时段内
- 建议：在14:50-14:59之间设置卖出（接近闭市）

### 日内交易
- 由于T+1规则，不能当日买卖
- 如果要"平仓"，应该：
  1. 设置卖出时间为次日
  2. 或者标记为"隔日平仓"
  3. 或者记录为"当日取消交易"（发现错误立即纠正）

### 记录规范
- 所有交易时间应在交易时段内
- 如发现错误，记录为"发现时间X:XX，立即纠正，标记为当日取消"
- 模拟交易时间应与实际市场时间一致
