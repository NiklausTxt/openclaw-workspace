## 🎯 二、交易风格选择

### 主风格：**短线题材博弈**

**选择理由**：
1. ✅ 政策驱动明显（政府工作报告未来产业概念）
2. ✅ 热点板块清晰（脑机接口、电力、科技）
3. ✅ 市场情绪回暖（超4300只个股上涨）
4. ✅ 创业板、科创50强势（成长股机会多）
5. ⚠️ 缩量上涨不适合长期持仓
6. ⚠️ 热点轮动快，需快速应对

### 辅助风格：**趋势波段**
- 适用于趋势明确的板块（如电力）
- 配合短线策略，提高胜率

## 💰 三、资金管理方案

### 3.1 仓位控制规则

| 仓位类型 | 规则 | 说明 |
|----------|------|------|
| 单票最大仓位 | ≤30% | 风险分散，避免集中持仓 |
| 总仓位（进攻） | 60-80% | 市场强势时高仓位 |
| 总仓位（稳健） | 40-60% | 量能不足时降仓位 |
| 总仓位（防守） | 20-40% | 风险高时低仓位 |
| 最低仓位 | ≥20% | 保持市场敏感度 |

**当前市场仓位建议**: **40-60%（稳健仓位）**
- 理由：缩量上涨，谨慎乐观

### 3.2 风险控制

| 风险指标 | 警戒线 | 当前值 | 状态 |
|----------|--------|--------|------|
| 单笔最大亏损 | 3% | 0% | ✅ 正常 |
| 总账户回撤 | 10% | 0% | ✅ 正常 |
| 单票最大仓位 | 30% | 0% | ✅ 正常 |
| 连续亏损次数 | 3笔 | 0笔 | ✅ 正常 |

### 3.3 止损止盈策略

#### 固定止损
- **短线交易**: -5%
- **趋势波段**: -8%
- **触发后**: 立即止损，不犹豫

#### 移动止损（趋势波段）
- **启动条件**: 持仓盈利≥8%
- **止损线**: 成本价+盈利×50%
- **调整频率**: 每日收盘后调整

#### 止盈策略
- **短线交易**:
  - 第一目标: +8%（减仓50%）
  - 第二目标: +15%（全部止盈）
- **趋势波段**:
  - 破位止盈: 跌破10日均线或关键支撑位
  - 技术止盈: MACD顶背离、量价背离

## 🔍 四、股票筛选规则

### 4.1 基本筛选条件

| 条件 | 短线题材 | 趋势波段 |
|------|----------|----------|
| 市值区间 | 50-500亿 | 100-1000亿 |
| 日成交量 | ≥5亿 | ≥10亿 |
| 换手率 | 5-15% | 3-10% |
| 涨幅要求 | -3%~+5%（不追高） | 趋势向上 |

### 4.2 短线题材筛选标准

#### 核心板块（优先级排序）
1. **脑机接口** 🥇
   - 政策驱动：政府工作报告明确提及
   - 板块涨幅：+3.25%
   - 龙头：岩山科技（涨停）、南京熊猫（+9%）
   - 逻辑：未来产业，政策大力支持

2. **电力** 🥈
   - 突破形态：桂冠电力直线涨停
   - 板块涨幅：+2.39%
   - 逻辑：经济复苏+政策支持

3. **CPO/算力租赁** 🥉
   - 科技主线：新易盛成交额200亿（+11.73%）
   - 板块活跃：MicroLED、液冷服务器等联动
   - 逻辑：AI硬件需求持续

4. **化工板块**
   - 期现联动：纯苯、苯乙烯涨停
   - 风险：波动较大，需谨慎

#### 选股标准
- ✅ 政策驱动（脑机接口、未来产业）
- ✅ 成交量活跃（日成交≥5亿）
- ✅ 涨幅合理（-3%~+5%，不追高）
- ✅ 有龙头效应（板块有涨停股）
- ❌ 避免涨幅>8%的个股（风险高）
- ❌ 避免ST、退市风险股

### 4.3 趋势波段筛选标准

#### 核心条件
- ✅ 趋势向上：股价位于20日均线之上
- ✅ 成交量稳定：日均成交≥10亿
- ✅ 基本面良好：营收、利润增长
- ✅ 板块共振：同板块多只个股上涨

#### 重点标的（待筛选）
- 电力龙头：桂冠电力、华银电力、上海电力
- 科技龙头：需进一步筛选

## 📋 五、交易执行流程

### 5.1 每日执行流程（交易时段）

#### 开盘前（09:00-09:25）
1. 查看隔夜美股、期指走势
2. 阅读重要新闻、政策公告
3. 复盘昨日持仓，检查止损止盈
4. 筛选当日目标股票池（5-10只）

#### 开盘后（09:30-10:30）
1. 观察板块开盘表现
2. 快速筛选强势个股
3. 等待回调机会，不追高
4. 记录买入逻辑、止损位、目标位

#### 交易时段（10:30-14:50）
1. 监控持仓变动
2. 检查止损止盈触发
3. 关注盘中异动
4. 记录交易决策

#### 收盘前（14:50-15:00）
1. 检查持仓状态
2. 执行必要的止盈止损
3. 更新持仓记录
4. 准备次日交易计划

#### 收盘后（15:00-17:00）
1. 更新 capital_status.md
2. 更新 daily_log.md
3. 记录交易文档
4. 复盘当日操作
5. 制定次日计划

### 5.2 买入决策流程

```
1. 筛选股票池 → 2. 确认板块趋势 → 3. 检查个股形态
↓
4. 确认买入时机（不追高） → 5. 计算仓位 → 6. 设置止损止盈
↓
7. 执行买入 → 8. 记录交易文档
```

### 5.3 卖出决策流程

```
1. 触发止损 → 立即止损
2. 触发止盈 → 分批止盈
3. 趋势破位 → 止盈离场
4. 更新交易文档
```
