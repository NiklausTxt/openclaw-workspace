# 建军照片自动同步方案

## 目标
自动检测"建军"共享相册的新照片，自动提取、分析和处理，为小红书运营提供素材。

---

## 方案概述

### 核心流程
```
定时任务(每小时) → 截图照片应用 → AI识别新照片 → 导出/处理 → 更新工作目录 → 提示用户
```

---

## 技术方案

### 阶段1：监控与识别（当前可行）

#### 1.1 定时检测
- **工具**: cron
- **频率**: 每小时执行一次
- **任务**: 调用监控脚本

#### 1.2 截图识别
- **工具**: peekaboo (截图照片应用)
- **AI**: image 工具 (识别照片数量、内容)
- **逻辑**:
  1. 截取"照片"应用的"建军"相册界面
  2. AI 识别当前显示的照片数量
  3. 与上次记录对比
  4. 如果有新增，触发处理流程

#### 1.3 状态记录
- **文件**: `jianjun/album_state.json`
- **内容**:
  ```json
  {
    "last_photo_count": 0,
    "last_check_time": "2026-03-03T11:30:00+08:00",
    "new_photos_since_last": [],
    "total_synced": 0
  }
  ```

---

### 阶段2：自动导出（需要探索）

#### 方案A: 使用 osascript (AppleScript)
```bash
# 通过 AppleScript 控制照片应用导出照片
osascript -e 'tell application "Photos"
  -- 选择共享相册中的照片
  -- 导出到指定目录
end tell'
```

#### 方案B: 使用快捷指令 (Shortcuts)
- 创建一个快捷指令：导出建军相册最新照片
- 通过命令行调用快捷指令
- 快捷指令可能支持 iCloud 共享相册访问

#### 方案C: 直接访问文件系统
```bash
# 检查照片库中的共享相册目录
ls -la ~/Pictures/Photos\ Library.photoslibrary/Resources/derivatives/
```

---

### 阶段3：智能分析（完全可行）

#### 3.1 照片分析
- **工具**: image 工具
- **内容**:
  - 识别照片场景（睡觉、玩耍、吃饭、洗澡...）
  - 识别表情（可爱、呆萌、活泼...）
  - 识别姿势（躺、坐、跳、伸懒腰...）
  - 评估照片质量（清晰度、构图、光线）

#### 3.2 标签化
- 为每张照片添加标签
- 存储到 `jianjun/photos_metadata.json`
- 方便后续检索和筛选

---

### 阶段4：小红书运营（完全可行）

#### 4.1 内容策划
- 根据新照片自动生成笔记主题
- 匹配已有内容策略

#### 4.2 图片处理
- 从照片应用导出高质量图片
- 或截图+裁剪

#### 4.3 文案生成
- 自动生成小红书笔记文案
- 包含标题、正文、话题标签

## 通知机制

### 发现新照片时
1. 发送飞书通知："建军相册有 [X] 张新照片"
2. 简要描述照片内容
3. 询问是否需要处理

### 处理完成时
1. 发送飞书通知："已完成 [X] 张照片的分析"
2. 提供处理结果摘要
3. 附上可能的小红书笔记建议
